    • Introduction

    Welcome to the GetSocial API! You can use our API to access GetSocial API endpoints, which provides data access to sharing statistics of news articles in public social networks & private messaging apps.

    Authentication

    To authorize, use this code:

    # With shell, you can just pass the correct header with each request
curl "api_endpoint_here"
  -H "Authorization: 4580df25599d5baf7fdd"

    Make sure to replace 4580df25599d5baf7fdd with your API key.

    GetSocial uses API keys to allow access to the API.

    GetSocial expects for the API key to be included in all API requests to the server in a header that looks like the following:

    Authorization: 4580df25599d5baf7fdd

    You must replace 4580df25599d5baf7fdd with your personal API key.

    Stories

    Get Top Stories

    curl "analytics.getsocial.io/api/stories/top?limit=3&metric=referrals"
  -H "Authorization: 4580df25599d5baf7fdd"

    The above command returns JSON structured like this:

    [
  {
    "identifier": "75dfd2a4fb79",
    "path": "/lyfestyle/article-z",
    "title": "Article z",
    "shares": 40,
    "referrals": 150
  },
  {
    "id": "f15a173dd497",
    "name": "/sport/article-x",
    "title": "Article x",
    "shares": 50,
    "referrals": 110
  },
  {
    "id": "5f341a5d899a",
    "name": "/sport/article-y",
    "title": "Article y",
    "shares": 100,
    "referrals": 100
  }
]

    This endpoint retrieves the top stories in the last 24 hours sorted by a specific metric. It can be used to retrieve the mosted shared or most referred stories.

    HTTP Request

    GET http://analytics.getsocial.io/api/stories/top

    Query Parameters

    Parameter Type Default Description Required
    limit integer 10 Number of stories to be retrieved No
    metric string shares Metric used to sort stories. Accepted values are shares and referrals No

    Returns

    List of stories sorted by metric.

    Errors

    The GetSocial API uses the following error codes:

    Error Code Meaning
    400 Bad Request -- Your request sucks.
    401 Unauthorized -- Your API key is wrong.
    406 Not Acceptable -- You requested a format that isn't json.
    500 Internal Server Error -- We had a problem with our server. Try again later.
    503 Service Unavailable -- We're temporarily offline for maintenance. Please try again later.
